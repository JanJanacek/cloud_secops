# cloud_secops

A hands-on learning repository documenting my transition toward a **Cloud Security Engineer** role.

---

## About this repository

This repository is part of my **12-month learning roadmap** focused on building practical cloud security skills, with **AWS** as the primary platform.

The goal is not just to study theory, but to complete labs, document findings, validate security concepts in practice, and build a portfolio of meaningful security-focused projects.

---

## What I am focusing on

Over time, this repository will cover areas such as:

- **AWS security foundations**
- **IAM and least privilege**
- **Cloud networking basics for security**
- **Logging, monitoring, and visibility**
- **Terraform and secure Infrastructure as Code**
- **AWS security services and controls**
- **S3, EC2, CloudTrail, KMS, and Config security**
- **Container security**
- **Kubernetes security basics**
- **CI/CD security basics**
- **SIEM, log analysis, and basic detection engineering**
- **Threat and risk thinking in cloud environments**

---

## Repository purpose

I use this repository to:

- document hands-on labs
- store notes and summaries
- track security experiments and findings
- keep evidence of completed work
- build a portfolio for future cloud security opportunities

---

## Repository structure

The content of this repository will grow over time and may include:

- `labs/` – hands-on lab work and setup notes
- `notes/` – study notes and concept summaries
- `projects/` – mini projects and portfolio work
- `terraform/` – Infrastructure as Code experiments
- `detections/` – detection ideas and log analysis exercises
- `reviews/` – security reviews, findings, and lessons learned
